## 디자인 / 기술

- 단일 HTML (외부 JS 라이브러리 의존 없음, Pretendard CDN만 사용)
- 인터랙티브 SVG 차트 — 시나리오 토글, 호버 가이드, 데이터 포인트 활성화, 라인 drawing 애니메이션
- 배경 — CSS 그라데이션 메쉬 + Canvas 도시 불빛 파티클 + SVG 그레인 노이즈
- 한국어 우선 (Pretendard Variable + 시스템 폴백)
- `prefers-reduced-motion` 존중
